## Requirements:

```sh
1 Download & Install: XAMPP in C:\xampp (default)
2 Clone this repository in C:\xampp\htdocs
3 Run XAMPP and start "Apache" and "MySQL"
4 Open the link "localhost/phpmyadmin/"
5 Click on new at sidebar and create a database name "bluebirdhotel"
6. After clicking database click import and select the file "bluebirdhotel.sql"
7. At the terminal in VScode, enter 'npm i bootstrap@5.3.2' to install bootstrap 
Open the link "http://localhost/homeNoLogin.php/" (without register) or "http://localhost/index.php/" (register / login) 
Now register and login
```
